API desenvolvida para sustentar o ecossistema do **Horizon Barber**. Responsável pela lógica de negócios, autenticação de usuários e persistência de agendamentos e serviços.
Atualmente a API funciona totalmente online e é consumida pelo projeto [Horizon Barber](https://github.com/Samuel-millerr/horizon-barber).
## 🚀 Funcionalidades

- **Auth:** Cadastro e autenticação de usuários.
- **Services:** Listagem de serviços oferecidos pela barbearia.
- **Appointments:** CRUD de agendamentos (Create, Read, Delete).
- **Users:** Atualização de perfil (PATCH).

---

## 🛠️ Tecnologias

- **Linguagem:** Python 3.4.2
- **Framework:** [FastAPI](https://fastapi.tiangolo.com/)
- **ORM:** [SQLAlchemy](https://www.sqlalchemy.org/)
- **Banco de Dados:** SQLite
- **Deploy:** [Render](https://render.com/)
